# final_project_nypd

This folder contains a .Rmd file and, as a reference, a pdf version of the knitted output that should be produced.

# To run the .Rmd :

Recommended Prerequisites: 
1. R 
2. RStudio
3. necessary R packages (Knitting the .Rmd in RStudio should load the libraries and/or prompt you to install any missing packages, but the packages do need to be installed first)

Run the File with RStudio:
1. Download the .Rmd file
2. Open it in RStudio
3. Click Knit in RStudio
4. The knitted document should show in your project folder
